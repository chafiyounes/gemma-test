Documents RAG

Guide rapide · corpus et brouillon
  • Ajouter des fichiers : cliquez sur la zone ci-dessous ou glissez-déposez. Pas de bouton « parcourir » séparé : le clic ouvre le même sélecteur.
  • Formats : .md (copié tel quel, idéal pour liens publics via /api/rag-media/<catégorie>/…), .txt, .docx (le serveur génère aussi un .md à l’enregistrement si possible).
  • Actualiser la liste recharge l’aperçu depuis le disque (utile après une modif externe ; n’applique pas le brouillon).
  • Un corpus = un dossier sur le disque (nom de catégorie). Le chat et l’agentic utilisent les mêmes noms.
  • Champ Corpus cible vide → fichiers vers RAG_DEFAULT_CATEGORY (souvent procedures) dans .env.
  • Les changements restent en brouillon jusqu’à Enregistrer sur le disque.
  • Gros lots : une seule requête ; le proxy peut limiter la taille totale.
Laisser vide pour utiliser le corpus par défaut du serveur.

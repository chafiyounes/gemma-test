Pour créer votre compte chez SENDIT, veuillez suive les étapes suivantes :

1. **Accéder au Site Web :** Rendez-vous sur le site web de SENDIT <https://www.sendit.ma/>
2. **Cliquer sur S'inscrire.**

   [![](https://downloads.intercomcdn.com/i/o/811141816/f07d6f7ccbb19c8fb5a32f78/fffff.gif?expires=1778668200&signature=61dffeb1711492d2f94d480715a83fa946cd1d36736ca29fe4e1803edf4070a4&req=fCEmF81%2FlYBZFb4f3HP0gHRFP9PUaGrCxnb%2BTSTU7uao5ObNHa8w1QTD2LJx%0ATUs%3D%0A)](https://downloads.intercomcdn.com/i/o/811141816/f07d6f7ccbb19c8fb5a32f78/fffff.gif?expires=1778668200&signature=61dffeb1711492d2f94d480715a83fa946cd1d36736ca29fe4e1803edf4070a4&req=fCEmF81%2FlYBZFb4f3HP0gHRFP9PUaGrCxnb%2BTSTU7uao5ObNHa8w1QTD2LJx%0ATUs%3D%0A)
3. **Remplir le formulaire aves vos données.**

   [![](https://downloads.intercomcdn.com/i/o/811142248/f9c1640f978ad8b452699834/image.png?expires=1778668200&signature=07d52c67fcff54dbe19b730a2cccd49433d2e97691a669c6504b9b6bf68040b4&req=fCEmF818n4VXFb4f3HP0gPD%2FN2FxqqTiByqWorfcEAKZKjxBc7PWAD1gxeRP%0A4mw%3D%0A)](https://downloads.intercomcdn.com/i/o/811142248/f9c1640f978ad8b452699834/image.png?expires=1778668200&signature=07d52c67fcff54dbe19b730a2cccd49433d2e97691a669c6504b9b6bf68040b4&req=fCEmF818n4VXFb4f3HP0gPD%2FN2FxqqTiByqWorfcEAKZKjxBc7PWAD1gxeRP%0A4mw%3D%0A)

   Il serait nécessaire de préciser les éléments suivant :

   * **Nom**
   * **Prénom**
   * **Adresse mail**
   * **Mot de passe**
   * **Téléphone**
   * **Nom de la marque** : ce nom sera généré sur les étiquettes de vos colis, il serait nécessaire d'indiquer le bon nom de votre marque.
   * **Votre site ou page Facebook/Instagram** : afin de faciliter la gestion de votre compte merci de nous fournir le lien de votre page Facebook/Instagram.
4. **Cliquez sur s'inscrire**

### Une fois que notre équipe technique aura validé la demande, vous devriez recevoir un message SMS ainsi qu'un courriel pour confirmer la création de votre compte.

Pour réinitialiser votre mot de passe, merci de suivre les étapes suivantes :

1. Cliquer sur "[Mot de passe oublié ?](https://app.sendit.ma/forget-password) " sur le lien de notre plateforme : <https://app.sendit.ma/login>.

   [![](https://downloads.intercomcdn.com/i/o/813132181/275eb5f318da3c9140b14cb1/image.png?expires=1778668200&signature=dbd6bc29b8ae1c46fbce94777568b77da03536094aced14913e8fdacd75bbdba&req=fCEkF8p8nIleFb4f3HP0gCSIKh4h2QP2Pz0YNYT38XWcpTA1399QS18hlc48%0Af3U%3D%0A)](https://downloads.intercomcdn.com/i/o/813132181/275eb5f318da3c9140b14cb1/image.png?expires=1778668200&signature=dbd6bc29b8ae1c46fbce94777568b77da03536094aced14913e8fdacd75bbdba&req=fCEkF8p8nIleFb4f3HP0gCSIKh4h2QP2Pz0YNYT38XWcpTA1399QS18hlc48%0Af3U%3D%0A)
2. Saisir votre adresse mail et cliquer sur **envoyer**.

   [![](https://downloads.intercomcdn.com/i/o/813132181/275eb5f318da3c9140b14cb1/image.png?expires=1778668200&signature=dbd6bc29b8ae1c46fbce94777568b77da03536094aced14913e8fdacd75bbdba&req=fCEkF8p8nIleFb4f3HP0gCSIKh4h2QP2Pz0YNYT38XWcpTA1399QS18hlc48%0Af3U%3D%0A)](https://downloads.intercomcdn.com/i/o/813132181/275eb5f318da3c9140b14cb1/image.png?expires=1778668200&signature=dbd6bc29b8ae1c46fbce94777568b77da03536094aced14913e8fdacd75bbdba&req=fCEkF8p8nIleFb4f3HP0gCSIKh4h2QP2Pz0YNYT38XWcpTA1399QS18hlc48%0Af3U%3D%0A)
3. Un message de confirmation apparaîtra.

   [![](https://downloads.intercomcdn.com/i/o/814978987/3a739d0700fc793a9fb30d29/image.png?expires=1778668200&signature=eeff4518321f1cd4711467e1e55b2d5cc44dbed6da0f8f56cfb02d3ae90a0b5e&req=fCEjH852lIlYFb4f3HP0gCz5FIv45OQK0ujXWcacxhTmn6Swpz5ySRNxN09y%0Ab24%3D%0A)](https://downloads.intercomcdn.com/i/o/814978987/3a739d0700fc793a9fb30d29/image.png?expires=1778668200&signature=eeff4518321f1cd4711467e1e55b2d5cc44dbed6da0f8f56cfb02d3ae90a0b5e&req=fCEjH852lIlYFb4f3HP0gCz5FIv45OQK0ujXWcacxhTmn6Swpz5ySRNxN09y%0Ab24%3D%0A)
4. Vous devez recevoir un mail pour modifier le mot de passe. Cliquer sur "**Réinitialiser le mot de passe**".

   [![](https://downloads.intercomcdn.com/i/o/814983317/19385593298010038b11d1c6/image.png?expires=1778668200&signature=f6080ac48c68802a1b738d999941e9e8806daa849a8ba83531521de29d2599ef&req=fCEjH8F9noBYFb4f3HP0gA8p1j52aB%2F2TBOKabpUaOgf7VV0SuXZOX8Op5bV%0A8pQ%3D%0A)](https://downloads.intercomcdn.com/i/o/814983317/19385593298010038b11d1c6/image.png?expires=1778668200&signature=f6080ac48c68802a1b738d999941e9e8806daa849a8ba83531521de29d2599ef&req=fCEjH8F9noBYFb4f3HP0gA8p1j52aB%2F2TBOKabpUaOgf7VV0SuXZOX8Op5bV%0A8pQ%3D%0A)
5. Saisir votre adresse mail et le nouveau mot de passe puis cliquer sur "**Réinitialiser**"

   [![](https://downloads.intercomcdn.com/i/o/814984708/b8fb2efa950fa431d661b5a0/image.png?expires=1778668200&signature=a6c2d88526742bdab2fdefba544d785181e157b4013917b935c5719595cf4d02&req=fCEjH8F6moFXFb4f3HP0gOFKmZmWqgVLoYf1wuREH6Sop8MRWdY6JI9bxaFR%0AMvk%3D%0A)](https://downloads.intercomcdn.com/i/o/814984708/b8fb2efa950fa431d661b5a0/image.png?expires=1778668200&signature=a6c2d88526742bdab2fdefba544d785181e157b4013917b935c5719595cf4d02&req=fCEjH8F6moFXFb4f3HP0gOFKmZmWqgVLoYf1wuREH6Sop8MRWdY6JI9bxaFR%0AMvk%3D%0A)
